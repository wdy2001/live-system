# 生活缴费系统 - Product Requirement Document

## Overview
- **Summary**: 一个集电费、水费、燃气费缴费与管理于一体的生活服务系统，支持在线缴费、缴费记录查询、计费规则展示和故障报修功能。
- **Purpose**: 为居民用户提供便捷的一站式生活缴费服务，解决传统线下缴费流程繁琐、效率低的问题。
- **Target Users**: 居民家庭用户、物业管理人员

## Goals
- 实现电费、水费、燃气费的在线缴费功能
- 提供分类查看缴费记录的功能
- 展示各类费用的计费规则
- 提供故障报修提报和查询功能
- 系统架构清晰，代码可维护性高
- 支持部署到 GitHub 仓库

## Non-Goals (Out of Scope)
- 不实现真实支付网关对接，使用模拟支付流程
- 不实现用户实名认证和身份验证
- 不实现多租户/多小区管理
- 不实现移动端 App，仅提供 Web 端
- 不实现发票开具功能
- 不实现管理员后台的精细化权限管理

## Background & Context
- 传统生活缴费需要用户分别到不同营业厅或使用多个 App 完成，操作繁琐
- 本系统旨在统一水电燃气缴费入口，提升用户体验
- 采用前后端分离架构，便于后续扩展和维护

## Functional Requirements
- **FR-1**: 用户注册与登录 - 支持用户账号注册、登录、退出
- **FR-2**: 电费缴费 - 用户可查询电费账单并完成缴费
- **FR-3**: 水费缴费 - 用户可查询水费账单并完成缴费
- **FR-4**: 燃气费缴费 - 用户可查询燃气费账单并完成缴费
- **FR-5**: 缴费记录查询 - 按费用类型（电/水/燃气）分类查看历史缴费记录
- **FR-6**: 计费规则展示 - 展示电、水、燃气的阶梯计费规则
- **FR-7**: 故障报修 - 用户可提交故障报修工单，查看报修进度
- **FR-8**: 用户中心 - 查看和编辑个人信息、绑定户号

## Non-Functional Requirements
- **NFR-1**: 性能 - 页面加载时间 < 2秒，API 响应时间 < 500ms
- **NFR-2**: 安全性 - 用户密码加密存储，接口需要鉴权
- **NFR-3**: 可用性 - 界面简洁友好，操作流程清晰
- **NFR-4**: 可维护性 - 代码结构清晰，有良好的注释和文档
- **NFR-5**: 兼容性 - 支持主流浏览器（Chrome、Firefox、Safari、Edge）

## Constraints
- **Technical**: 
  - 后端：Python (Flask/FastAPI)
  - 前端：Vue.js 3 + Vite
  - 数据库：MySQL 8.0+
  - ORM：SQLAlchemy
- **Business**: 
  - 开发周期：10天
  - 功能范围限定为核心缴费功能
- **Dependencies**: 
  - MySQL 数据库服务
  - GitHub 仓库用于代码托管

## Assumptions
- 用户户号预先存在于系统中，用户通过户号绑定自己的账户
- 缴费为模拟支付流程，不涉及真实资金流转
- 故障报修工单状态为：待处理、处理中、已完成
- 计费规则为固定阶梯价格，由系统预置

## Acceptance Criteria

### AC-1: 用户注册与登录
- **Given**: 用户访问系统登录页
- **When**: 用户输入有效的用户名和密码进行注册，然后使用注册的账号登录
- **Then**: 注册成功后可登录系统，登录后跳转到首页，显示用户信息
- **Verification**: `programmatic`

### AC-2: 电费缴费
- **Given**: 用户已登录并绑定了电费户号
- **When**: 用户进入电费缴费页面，查看账单后点击缴费
- **Then**: 显示当前电费账单金额，缴费成功后余额扣减，生成缴费记录
- **Verification**: `programmatic`

### AC-3: 水费缴费
- **Given**: 用户已登录并绑定了水费户号
- **When**: 用户进入水费缴费页面，查看账单后点击缴费
- **Then**: 显示当前水费账单金额，缴费成功后余额扣减，生成缴费记录
- **Verification**: `programmatic`

### AC-4: 燃气费缴费
- **Given**: 用户已登录并绑定了燃气户号
- **When**: 用户进入燃气费缴费页面，查看账单后点击缴费
- **Then**: 显示当前燃气费账单金额，缴费成功后余额扣减，生成缴费记录
- **Verification**: `programmatic`

### AC-5: 分类查看缴费记录
- **Given**: 用户已登录并有多条缴费记录
- **When**: 用户进入缴费记录页面，选择按类型筛选（全部/电费/水费/燃气费）
- **Then**: 列表显示对应类型的缴费记录，包含时间、金额、类型、状态等信息
- **Verification**: `programmatic`

### AC-6: 计费规则展示
- **Given**: 用户访问计费规则页面
- **When**: 用户查看电、水、燃气的计费规则
- **Then**: 分别展示电、水、燃气的阶梯计价规则，包含阶梯区间和单价
- **Verification**: `programmatic`

### AC-7: 故障报修
- **Given**: 用户已登录
- **When**: 用户填写故障报修单（类型、描述、地址、联系方式）并提交
- **Then**: 报修单提交成功，可在报修记录中查看工单状态
- **Verification**: `programmatic`

### AC-8: 用户中心
- **Given**: 用户已登录
- **When**: 用户进入个人中心，修改个人信息或绑定/解绑户号
- **Then**: 个人信息更新成功，户号绑定/解绑生效
- **Verification**: `programmatic`

### AC-9: UI 友好性
- **Given**: 用户使用系统各功能页面
- **When**: 用户浏览和操作系统
- **Then**: 界面布局美观、交互流畅、提示信息清晰
- **Verification**: `human-judgment`
